# Repetition: JavaScript

I dagens laboration kommer du att få lära dig grunderna i JavaScript-programmering. Vi börjar med att gå igenom grunderna (variabler, loopar, funktioner m.m.) och går därefter vidare till datastrukturer och annat smått och gott.

**OBS - Det är många övningar**, det kan vara klokt att inte göra allt på en gång. Räkna med att det totalt kommer att **minst 4 timmar** att gå igenom alla övningar för JavaScript här - så planera din tid och börja gärna så fort som möjligt - varför inte kicka igång innan kursen drar igång för att få en flygande start? 😊
{:.info}

Laborationen består av ett gäng interaktiva och väldigt genomtänkta uppgifter av [Beau Carnes](http://carnes.cc/), som är bättre på att göra JavaScript-kurser än Johan. Uppgifterna hittar ni på [freeCodeCamp](https://www.freecodecamp.org/learn/javascript-v9/). Nedan finns en bra genomgång av JavaScript för den som programmerat tidigare (alltså ni).
