# Repetition: HTML/CSS

I dagens laboration kommer du att få lära dig grunderna i HTML & CSS. Vi börjar med att gå igenom grunderna i HTML och går därefter vidare till grunderna i CSS och avslutar med att titta på tillgänlighet för information på webben.

**OBS - Det är många övningar**, det kan vara klokt att inte göra allt på en gång. Räkna med att det totalt kommer att **minst 8 timmar** att gå igenom alla övningar för HTML & CSS här - så planera din tid och börja gärna så fort som möjligt - varför inte kicka igång innan kursen drar igång för att få en flygande start? 😊
{:.info}

Laborationen består av ett gäng interaktiva och väldigt genomtänkta uppgifter som ni hittar på [freeCodeCamp](https://www.freecodecamp.org/learn/responsive-web-design-v9/), som är bättre på att göra interaktiva HTML- och CSS-kurser än Anton. En perfekt plats för att repetera (och utöka) sina kunskaper inom HTML & CSS.

Tänk på att ni ska lämna in en skärmdump som visar på att ni är klara med uppgiften. Därför rekommenderar vi att ni **skapar ett konto på sidan** (dock frivilligt) som kommer ihåg er process.
{: .info}
